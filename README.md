# Portfolio of Procurement Work
This repository contains several documents that I have created in order to improve procurement processes.

# Price Review Form Template
This document is meant for analysing any price changes to baskets of goods. 
The main table is filled out with existing pricing before the supplier inputs any new pricing.
The hidden columns are then able to analyse the pricing for any changes.

# Price Checking Dashboard Template
This document is meant to check invoice data against system pricing to check for any potential overcharges.
Invoice data is pasted into the main sheet and then the pivot tables create a summary of the charges per product code ready for negotitation with the supplier.
Further pivots show analysis of the total amount of charges.

# Basket Review Template
This document is used to check the spend coverage of a basket of goods and help to identfity which products should be added or removed to increase the spend coverage.

# Procurement KPI Dashboard
This dashbaord is created to simplify very generic business KPIs that can be used to track company performance.
